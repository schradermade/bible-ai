# Core User Journeys (V1)

## 1. Purpose
Define the critical user flows that validate V1 and shape UX, AI behavior, and data needs.

---

## 2. Journey: Guided AI Entry (Home)

**User Goal**  
Start a Bible-guided AI interaction without facing a blank chat box.

**Flow**  
1. User lands on home
2. Sees guided options: Understand a passage, Think through a decision, Receive encouragement, Learn about prophecy
3. Selects an option
4. Sees a short prompt or input field with examples
5. Submits request
6. Receives structured AI response
7. Option to save response or start a new prompt

**Success Signal**  
User completes an AI interaction without confusion about what to ask.

---

## 3. Journey: Verse Explanation From Reading

**User Goal**  
Read Scripture and ask for help understanding a specific passage.

**Flow**  
1. User opens the Bible reader
2. Selects a passage
3. Chooses “Explain this” (plain meaning, context, original language insight, or application)
4. Receives AI response with citations and reflection question
5. Saves insight to notes or bookmarks

**Success Signal**  
User returns to reading with clearer understanding and saved insight.

---

## 4. Journey: Life Discernment

**User Goal**  
Think biblically about a real decision without being told what to do.

**Flow**  
1. User selects “Think through a decision”
2. Inputs a question or situation
3. AI reframes the question biblically
4. AI provides Scripture anchors and reflective prompts
5. User saves the response or adds a personal journal entry

**Success Signal**  
User reports the response was helpful and non-prescriptive.

---

## 5. Journey: Daily Encouragement

**User Goal**  
Receive calm Scripture-based encouragement.

**Flow**  
1. User selects “Receive encouragement”
2. Optionally selects a feeling or situation
3. AI returns a short Scripture-based response
4. Optional prayer prompt
5. User saves or shares

**Success Signal**  
User engages with a short, calm response and returns on future days.

---

## 6. Journey: Prophecy Introduction

**User Goal**  
Understand how Scripture approaches prophecy without sensationalism.

**Flow**  
1. User selects “Learn about prophecy”
2. AI provides a short framing response
3. User selects a passage or theme
4. AI explains the passage with humility and multiple views
5. User saves insights or bookmarks passages

**Success Signal**  
User leaves with clarity and trust, not fear or speculation.
